# EstherAB QA Portfolio 
Este repositorio contiene mi portafolio como QA Tester Junior, donde aplico conocimientos en pruebas manuales, automatización, API Testing y validación de datos con SQL.

Soy egresada de la carrera de Diseño y Desarrollo de Simuladores y Videojuegos, con interés en la calidad de software y la validación de sistemas en entornos reales.

## Estructura del Portafolio
### Manual Testing
Diseño de casos de prueba, plan de pruebas, evidencias y reporte de bugs.

### Automation Testing
Automatización de pruebas con Cypress:
- Login
- Products
- Cart
- Checkout

### API Testing
Pruebas de API con Postman:
- Validación de endpoints de usuario
- Registro, login y eliminación
- Búsqueda de productos

### SQL for QA
Consultas SQL básicas aplicadas a validación de datos:
- SELECT, WHERE
- COUNT
-JOIN (INNER / LEFT)
- GROUP BY
- ORDER BY
  
### Scrum
Gestión básica de proyectos ágiles con Jira.

## Proyecto base

Todas las pruebas están basadas en el sitio:
Automation Exercise: https://automationexercise.com/ 

## Objetivo
Demostrar habilidades prácticas en QA Manual, API Testing, Automatización y validación de datos, simulando escenarios reales de un entorno de e-commerce.

## Tecnologías
- Cypress
- Postman
- SQL (SQLite)
- Jira
- GitHub
