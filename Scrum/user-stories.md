# User Stories

## US-001 Registro de usuario
Como cliente
Quiero registrarme en la plataforma
Para poder realizar compras.

### Criterios de aceptación
- El usuario puede crear una cuenta.
- Los campos obligatorios deben validarse.
- Debe mostrarse mensaje de registro exitoso.

-------------------------------------------------------------

## US-002 Inicio de sesión
Como cliente registrado
Quiero iniciar sesión en la plataforma
Para acceder al catálogo de productos y realizar compras

### Criterios de aceptación:
- El usuario debe ingresar nombre de usuario y contraseña.
- El sistema debe permitir acceso con credenciales válidas.
- El sistema debe mostrar mensaje de error con credenciales inválidas.
- Los campos usuario y contraseña son obligatorios.
  
-------------------------------------------------------------

## US-003 Visualizar catálogo
Como cliente
Quiero visualizar los productos disponibles
Para seleccionar artículos para comprar

### Criterios de aceptación:
Debe mostrarse el listado de productos.
Cada producto debe mostrar nombre.
Cada producto debe mostrar precio.
Cada producto debe mostrar descripción.

-------------------------------------------------------------

## US-004 Visualizar detalle de Producto
Como cliente
Quiero visualizar el detalle de un producto
Para conocer sus características antes de comprar

### Criterios de aceptación:
Debe mostrarse nombre del producto.
Debe mostrarse descripción.
Debe mostrarse precio.
Debe existir opción para regresar al catálogo.
